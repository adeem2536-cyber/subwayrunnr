# 🏃 Speed Runner

Speed Runner is a **modern endless runner mobile web game** inspired by **Subway Surfers**.
The player runs forward automatically while avoiding obstacles, collecting coins, and surviving as long as possible.

The game is built using **HTML, CSS, and JavaScript**, making it lightweight and playable directly in the browser on both **mobile and desktop devices**.

---

# 🎮 Gameplay

* The character runs forward automatically.
* Swipe or use arrow keys to change lanes.
* Jump over obstacles and barriers.
* Collect coins to increase your score.
* Speed gradually increases as the game progresses.
* The game ends if the player collides with an obstacle.

---

# 🕹 Controls

### 📱 Mobile

* Swipe **Left / Right** → Change lane
* Swipe **Up** → Jump
* Swipe **Down** → Slide

### 💻 Desktop

* **Arrow Left / Right** → Change lane
* **Arrow Up** → Jump
* **Arrow Down** → Slide

# 🚀 How to Run the Game

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in a web browser

No installation required.

# 🌐 Hosting the Game

You can host the game for free using **GitHub Pages**.

Steps:

1. Upload the project to **GitHub**
2. Open repository **Settings**
3. Go to **Pages**
4. Select branch **main**
5. Save and get your live game link
